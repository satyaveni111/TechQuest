What is TechQuest

The TechQuest is a web based quiz system where students from ECE can attend the examination of the multiple choice questions.
In this platform they can also justify thenselves after giving the examination as the result will come after answering definite questions
for each set in each subject in each Domain.There is also an option for admin who can edit questions and answers.Moreover,the
admin can add or remove questions and can also remove a set.

Working of TechQuest

The homepage introduces the website, its purpose, and benefits. Users can see a brief
description of what the platform offers, like quizzes on various ECE subjects, along with
a call-to-action (e.g., “Sign Up” or “Login to Start Quizzing”).The login page is designed
for simplicity and user - friendliness, ensuring that users can quickly access of account
with minimal friction. The page includes: Name & Email. If the email related to RGUKT
mail then the user can access to start further process, otherwise user not allow to take
quiz. After successfully logging in, user are teken to adomain selection page. The page
includes list of ECE domains. Once a domain is selected, there are presented with the
subdomains related to that topic. After selecting the subdomain, users are promted to
choose the difficulty level of the quize. There are levels of basic, medium, hard. Once
difficulty level selected , the quize begins and user presented with series of questions
with some time limit. Ater answering all questions or out-off time, users are ended with
scorecard to summarizing their performance.
